## ğŸš€ Getting Started

### Prerequisites

- Node.js 16.14.0 or later
- npm (v7+) or yarn
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mvaid1990/syncbyte.git
   cd syncbyte
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
